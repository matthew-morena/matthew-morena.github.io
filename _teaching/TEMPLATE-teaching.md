This collection feeds the "Teaching" section of your CV page (_pages/cv.md),
which loops over `site.teaching`. It is separate from _pages/teaching.md,

1. Copy this file and rename it, e.g. 2026-spring-honr310.md
2. Fill in the front matter above (title, type, venue, date, location, permalink).

4. Write a short description below the front matter if you want one.
